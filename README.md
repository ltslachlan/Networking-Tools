# Networking Tools

Scripts for network diagnostics, troubleshooting, and configuration validation.

## 📌 Purpose
This repository contains tools to assist with identifying and resolving network-related issues, including DNS resolution, connectivity checks, and basic monitoring setup.

## 🛠 Scripts Included
- DNS and nslookup automation
- Connectivity and latency checks
- SNMP configuration helpers
- Network device validation scripts

## 🚀 Usage
Run scripts locally or against remote systems depending on the tool:

```powershell
.\scriptname.ps1
